# MSML606-Project-2
MSML606 Extra Credit Project 2 Repository
Directions for running the file:
When you run the 'Extra Credit Project 2 Code'.py file, you will be prompted to select an origin city and destination city. This code uses the city and distance data in the 'Cities Graph Dataset'.csv file. Make sure the format of the cities you choose is "City#" when you type it, or else you will receive an error statement. After you select each city, the code will use dijkstra's algorithm to calculate the shortest path between your two cities, and will print the shortest distance and shortest path between the cities. If there is no path between the two cities you select, the algorithm will return infinity as the distance, and will print a statement stating that there is no path between the two cities.
